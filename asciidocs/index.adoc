==== Create a .puml-file

* in `<project-root>/asciidocs/plantuml`

[source, plantuml]
----
@startuml
class Person{}

class Hobby {}

Person <-- Hobby : übt aus
@enduml
----

[source,shell]
----
plantuml::plantuml/cld-demo.puml[]
